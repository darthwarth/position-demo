Base Element - No position
This entire parent element, contained within the double-lined border, has no positioning set within it and so it and its child elements just respect the normal page flow. The parent in this and each of the following cases contains two child elements; the title and the paragraph of text.

This is here to give a point of reference for the following examples.
Element One - Relative position
The paragraph in this example, which has a red dotted border, has a relative position, both to its top and its left of 20px. So it pushes it out of the normal page flow by these amounts.

The parent element is surrounded by a double border. The relative positioning of this paragraph of text pushes it out of its parent's border, as the parent is still the size it would be if the positioning was not set.
Element Two - Absolute position
The entire element in this example has had its absolute position set 200px from the top of the page and 20px from the left.

Using this setting will force the parent element to behave as though the child element isn't there, removing it from the normal page flow, hence why the positioning is now based on the page rather than the parent.

To make an absolutely positioned element position based on its parent rather than the page, set the parent's position to relative.
Element Three - Fixed position
Setting the fixed position on an element behaves in a similar way to the absolute property, except a fixed element is not affected by scrolling. As the rest of the non-fixed elements scrolled with the page, the fixed elements would stay as set.

The entire element in this example has had its fixed position set 450px from the top of the page and 20px from the right.
Element Four - Sticky position
Setting the position to sticky acts in a similar way to combining relative and fixed positions. In that the element will stay where it is until the position specified in the CSS is met, for example top: 50px.

In this particular example the title has is sticky position set to 80vh, so will remain where it is until that position is scrolled past, at which point the title element stays where it is on the screen while remaining within the parent element.
Demonstration - Mixing some styles together
In this mock profile card demo the image has a relative position set to -20px to both the top and the left, pushing it outside its container so it overlaps. The title has its position set to absolute by 5vh from the top. Which moves it down from the top of the containing border.

The overall card itself has its position set to fixed 50px from the bottom of the page and 50px from the right.
